Possibilidades com uso de JavaScript


Java Script e Java são a mesma coisa?
Java e JavaScript não são a mesma coisa. Embora os nomes sejam semelhantes, são linguagens de programação distintas com propósitos e características diferentes. Java: É uma linguagem de programação orientada a objetos de propósito geral, usada para criar uma ampla variedade de aplicativos, desde software para empresas até aplicativos móveis (como muitos apps para Android). É conhecida por sua portabilidade, o que significa que um programa Java pode ser executado em diferentes plataformas com poucas ou nenhuma modificação. JavaScript: É uma linguagem de programação interpretada, principalmente utilizada para criar interatividade em páginas da web. Ela permite que páginas web sejam mais dinâmicas, respondendo a ações do usuário e atualizando o conteúdo sem a necessidade de recarregar toda a página.


Quando surgiu o Java Script?
O JavaScript surgiu em 1995. Foi criado por Brendan Eich enquanto trabalhava na Netscape Communications Corporation, uma das pioneiras na área de navegadores web.A ideia era desenvolver uma linguagem de programação que permitisse criar páginas web mais dinâmicas e interativas, respondendo às ações do usuário sem a necessidade de recarregar toda a página. Inicialmente, a linguagem foi chamada de LiveScript, mas devido à popularidade da linguagem Java na época, foi renomeada para JavaScript.


Quem foi o responsável pela criação do Java Script?
O JavaScript foi criado por Brendan Eich na Netscape Communications Corporation em 1995. A Netscape era uma empresa de software que desenvolveu um dos primeiros navegadores web populares, o Netscape Navigator. A ideia era criar uma linguagem de script que permitisse aos desenvolvedores web adicionar mais interatividade às suas páginas. Inicialmente, a linguagem foi chamada de LiveScript, mas foi renomeada para JavaScript devido à crescente popularidade da linguagem Java na época.


Qual o caminho trilhado pelo reponsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?
Brendan Eich trabalhou na Netscape, depois ajudou a fundar a Mozilla e atualmente é CEO da Brave Software.
